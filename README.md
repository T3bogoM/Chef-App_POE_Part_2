# Chef-App_POE_Part_2

Chef Menu Builder App

A simple React Native app that allows chefs to create, view, and manage menus easily.
Users can add dishes with a name, description, course type, and price, then view or filter them.

Features include the following 
-Add new dishes with details
-View all dishes by course type
-Filter menu items by name or course
-Simple, colorful, and user-friendly UI

This app enables users to:
-Add dishes with details such as name, description, course type, and price
-View all dishes categorized by course (Starters, Mains, Desserts)
-Filter dishes based on course, name, or price
-Keep track of the total number of dishes added
-Navigate easily between screens using React Navigation

Required Packages for the app 
npm install @react-navigation/native @react-navigation/native-stack
npx expo install react-native-screens react-native-safe-area-context
